## NPM Deployment notes

* Open a terminal
* Navigate to the SRC folder
* Run "npm install"
* Run "npm run prepublish"
* Run "npm publish"

## Demo Development considerations

* Open a terminal
* Navigate to the SRC folder
* Run "npm link"
* Go back to the demo folder
* Run "npm link ng-chat"
* To unlink just run "npm unlink ng-chat"
